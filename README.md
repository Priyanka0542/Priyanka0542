## <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/People/Technologist.png" width="28" /> &nbsp;About Me

<img align="right" src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Objects/Laptop.png" width="120" />

**B.Tech Computer Science** student at **VIT-AP University** with a **9.64 / 10 CGPA**, driven by a simple belief: **technology should make people's lives better**.

Every project I build starts with a real problem — whether it's helping patients get faster clinical summaries, predicting health risks before they escalate, or giving people a private, judgment-free space to process their emotions. I don't build for the sake of building; I build because the right software, in the right hands, **can genuinely change outcomes for people**.

- 🧠 &nbsp;**Core Strengths:** Data Structures & Algorithms · Problem Solving · System Design
- 🔧 &nbsp;**What I Do:** Full-stack development, AI/ML model integration, and building privacy-first tools
- 🌍 &nbsp;**Philosophy:** Software should be **fast, private, accessible, and purposeful** — no exceptions
- 🤖 &nbsp;**ML Approach:** I leverage **vibe coding** with AI tools to rapidly prototype and ship ML-powered applications
- 🎯 &nbsp;**Currently:** Preparing for **campus placements** while sharpening competitive programming skills
